## Using Log 
Log display messages in the Logcat pane while running the app. Log messages are a powerful debugging tool that can be used to check on values, execution paths, and report exceptions.

![image](https://user-images.githubusercontent.com/58808609/100263914-34001880-2f76-11eb-91b0-8449d20150b4.png)

Log can be viewed by clicking the Logcat tab at the bottom of the Android Studio window.
